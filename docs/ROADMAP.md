# Roadmap

This file describes project-specific milestones.

It must not describe the evolution of this template.

## Milestone 1 — Project foundation

Goal: Merge upstream `docker-image-watcher` code with Forgejo-first agentic project setup.

Expected outcomes:

- Remotes configured for Forgejo origin and GitHub upstream.
- Project docs filled with real application context.
- GitNexus indexed and tooling verification started.
- Local-only config kept out of commits.

## Milestone 2 — First improvements

Goal: Implement first project-specific enhancements or fixes.

Expected outcomes:

- At least one scoped improvement or bug fix implemented.
- Relevant checks/tests documented and run.
- Documentation updated for changed behavior where needed.

## Milestone 3 — Upstream contribution

Goal: Prepare and send merge request or patch set upstream to original author.

Expected outcomes:

- Local changes kept understandable relative to upstream.
- Upstream-ready summary of improvements available.
- Contribution path to `redenfire/docker-image-watcher` validated.

## Deferred ideas

- Broader automation around scheduled update policies.
- Additional observability or notification integrations.
